Project Overview

The Burtday Gift project offers a fun web page with visual elements that respond to user interaction. The page features personalized birthday images, text shadows for emphasis, themed color schemes, and hover effects that reveal different images on specific elements. This interactive design provides a joyful experience for the user!

Features

  • Personalized Header: The header centers around a birthday greeting with a name and age display.
  • Colorful Background: The entire page has a soft, birthday-themed background color.
  • Hover Effects: Various images appear when users hover over specific keywords, adding an element of surprise.
  • Stylized Images: The project uses different images with frames and rounded borders to make the visuals engaging.
  • Fixed Footer: A fixed footer provides a persistent closing note or a signature at the bottom of the page.

Technologies Used

  • HTML & CSS: The project is built using basic HTML and CSS for layout and styling.
  • Custom Font: The "Happy Monkey" font is used for a playful, casual look.
  • Hover Effects: Adds interactivity with hover states on key elements.

Setup

  1. Clone the repository: 
    git clone https://github.com/your-username/Burtday-Gift.git
  2. Make sure all the images (e.g., Gift.jpg, Happy.webp, Handsome.webp, Genius.webp, Badass.webp, and Cheers.webp) are located in the images directory as specified in the CSS.
  3. Open the index.html file in a browser to view the project.

Usage

Once set up, simply open the page and hover over the highlighted words (such as "Happy," "Handsome," "Genius," etc.) to reveal images that enhance the celebratory feel of the page. The page is meant to create a joyful experience for a birthday or other celebratory event.
